# Day 11 Extension Task

This project runs a time-series EDA pipeline over the Sakshi PPG dataset.

## Run

Install the dependencies from this folder:

```powershell
python -m pip install -r requirements.txt
```

Run the pipeline from the `src` directory:

```powershell
Set-Location .\src
python main.py
```

## Outputs

The pipeline writes reports and plots to these folders in the project root:

- `reports/`
- `plots/`
- `decomposition/`
